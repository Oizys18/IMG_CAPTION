Python 경로 관리  [참고](https://itmining.tistory.com/122)

```python
import os
# 현재 작업 폴더 얻기 
os.getcwd()
# 디렉토리 변경
os.chdir('[path]')
# 특정 경로에 대해 절대 경로 얻기
os.path.abspath('[path]')
# 경로 중 디렉토리명만 얻기
os.path.dir(path)
# 경로 중 파일명만 얻기
os.path.basename(path)
```
